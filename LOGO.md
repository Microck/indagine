# Logo Prompts for Indagine

**Project Context**: A meta-agent system that debugs other AI agents when they fail. It captures failure events, analyzes traces and tool behavior, diagnoses root causes using a shared taxonomy, and proposes concrete fixes.

**Name Origin**: "Indagine" is Italian for "investigation" or "inquiry" — fitting for a tool that investigates why agents fail.

**Suggested Vibe**: Technical, intelligent, analytical, minimal, developer-focused

## Usage Notes

**Recommended for**: Midjourney, DALL-E 3, Ideogram, or similar AI image generators

**After generation**:
1. Select the strongest 2-3 concepts
2. Vectorize using Illustrator, Figma, or vectorizer.ai
3. Test at multiple sizes (favicon, 16px → app icon, 1024px)
4. Ensure works in single color (black/white)

**Color suggestions** (if adding color later):
- Primary: Deep blue (#1E3A5F) or teal (#0D7377) — trust, intelligence
- Accent: Amber (#F59E0B) or coral (#FF6B6B) — highlights, debugging alerts
- Consider: Purple gradient for AI/ML aesthetic if full color needed

**Filename convention**: `indagine-logo-[style]-[version].svg`
